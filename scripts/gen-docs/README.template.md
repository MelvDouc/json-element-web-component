# JsonElement

Create a customizable element out of JSON data.

## Usage

```ts
/* EXAMPLE */
```

outputs:

```html
<my-json-element>
  <dl>
    <dt>title</dt>
    <dd>The Three Musketeers</dd>
    <dt>publishedYear</dt>
    <dd>1844</dd>
    <dt>authors</dt>
    <dd>
      <ol>
        <li>Alexandre Dumas</li>
        <li>Auguste Maquet</li>
      </ol>
    </dd>
  </dl>
</my-json-element>
```
